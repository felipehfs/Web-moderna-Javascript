Introdução sobre HTML5

  • O HTML é case insensitive.
  • Possui possibilidade de criar suas próprias tags pelo framework.
  • Existem tags que não precisam ser fechadas, apenas em alguns frameworks Javascript.
  • Quando você escreve um HTML mais semântico isso ajuda na indexação deste conteúdo.
  • Informações sobre a páginas para os crawlers, metadados e importações de css geralmente fica no head da página.

Dentro de um HTML é possível integrar com CSS e Javascript, mas não é recomendado. O correto é utilizar uma folha de estilo e um script JS em um arquivo externo.

Introdução sobre o CSS

O CSS é responsável pela estilização na página utilizando customizando as tags pelos seletores e propriedades. Exemplo:
p { color: white; }
